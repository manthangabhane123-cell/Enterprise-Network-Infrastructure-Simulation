# Enterprise-Network-Infrastructure-Simulation
Enterprise campus network simulation using Cisco Packet Tracer featuring VLANs, Inter-VLAN Routing, OSPF, DHCP, DNS, Web &amp; Email Servers, Wireless Networking, Redundancy, and Network Security.

# Enterprise Network Infrastructure Simulation

## Project Overview

This project simulates a real-world enterprise campus network using Cisco Packet Tracer. The network includes multiple departments, redundant core switching, dynamic routing, centralized services, and wireless connectivity.
